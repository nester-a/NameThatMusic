1. Чтобы начать игру нужно нажать на кнопку Start Game, при этом появится дополнительное игровое поле, на котором игроки смогут наблюдать своё количество очков.
2. Чтобы кнопка Start Game стала активной, нужно сперва добавить на боле хотя бы одного игрока и загрузить директорию с музыкой:
	а. Для добавления игрока нажмите кнопку ADD. После добавления текст кнопки сменится на Active, а в окне игроков под аватаркой будет написано, что данный игрок в игре.
	б. Чтобы добавить диркторию с музыкой нажмите кнопку Options в верхнем меню. Напишите или выберите директорию с музыкой и нажмите кнопку Сохранить.
3. Кнопка начала проигрывания музыки станет активной только после начала игры, и будет продолжать оставаться активной пока все песни из директории не будут проиграны.
4. Во время проигрывания музыки кнопка STOP станет активной, также будет вестись обрабный отчёт таймера и рядом написано название текущей композиции.
5. Во время проигрывания музыки кнопка PLAY будет неактивной.
6. Как только все песни будут проиграны кнопки PLAY и STOP станут неактивны и рядом будет написано о том, что все песни из данной директории были проиграны.
7. Игрокам можно добавлять и убавлять очки за правильные и неправильные ответы соответственно нажатием кнопок + и -. Данные очки буду дублироваться из окна ведущего на окно для игроков. Очки нельзя сделать меньше нуля.

Возможности и ограничения, которые планируются реализовать записаны в текстовый файл Features.
Текущие проблемы записаны в файл Questions.

Данная версия уже более-менее пригодная для игры, поэтому присваиваю ей гит-версию 0.1.
